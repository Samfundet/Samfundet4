# Bilder

- Godkjente bildeformater: JPG, PNG, GIF, WebP, TIFF
- **Hele høyden** på et bilde er alltid synlig, mens sidene kan bli croppet basert
  på skjerm og kontekst. Hold alt som er viktig (i.e. tekst) mot midten av
  bildet
- Bilder for arrangementer bør lastes opp med en ratio på **4.5:1** (e.g. 3600×800 px).
